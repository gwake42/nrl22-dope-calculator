# NRL22 Dope Calculator

A mobile-first Progressive Web App (PWA) for calculating ballistic dope charts for NRL22 competitive shooting. Designed for match-day use with quick reference and printable dope cards.

- **Accurate Ballistic Calculations**: Sophisticated .22 LR ballistic engine with proper physics modeling
- **Ammo Presets**: Pre-configured for popular .22 LR rounds (Eley, SK, Lapua, CCI, Federal)
- **Weather Integration**: Auto-fetch current conditions via geolocation using Open-Meteo API
- **Stage Management**: Create unlimited stages with multiple target distances and shot sequences
- **Printable Dope Cards**: 2" x 3.5" cards optimized for scope caps - up to 20 cards per sheet
- **Quick Distance Lookup**: Instant dope calculation for any distance
- **PWA Support**: Install as native-feeling app on mobile devices
- **Offline Capable**: Works without internet connection once installed
